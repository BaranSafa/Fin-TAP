Current Project Features (v1.0)

  • Dynamic Backend API: A powerful API developed using Python Flask that handles all data processing and prediction requests.
  • Advanced Machine Learning: Gradient Boosting models using `XGBoost` / `LightGBM` to predict prices.
  • Comprehensive Feature Engineering: 25+ technical indicators (MACD, RSI, Bollinger Bands, Ichimoku Cloud etc.) calculated with `pandas-ta` library.
  • Multi-Page Modern Interface: Tailwind is a CSS-styled dashboard with dark mode, `Home`, `Predict`, `All Stocks` and `Suggestion` pages.
  • Automatic Model Training: Automatic training and saving of models for all stocks with the `train.py` script.
  • Smart Routing: Automatic `ticker` redirection from "All Stocks" page to "Predict" page and automatic start of the forecast.
  • Smart Suggestion System: Analysis module showing the top 3 stocks with the highest upside potential on the "Suggestion" page.
  • Desktop Application: Compiling the entire project into a single `.exe` file using `PyWebView` and `PyInstaller`.

Future Plans (v2.0)

  • "Prices" Page: By activating the "Prices" page, you can show real-time price data using `yfinance` or a different API.
  • Advanced "All Stocks" Page: Adding additional information such as company names and market capitalization to the table on the "All Stocks" page.
  • User Account: A “Portfolio” system where users can save their favorite stocks.
  • More Models: Integrating different time series models such as LSTM and (S)ARIMA and providing model selection feature on the "Predict" page.
  • Automatic Retraining: Setting up a CI/CD pipeline that automatically retrains models on a server on a weekly basis.
